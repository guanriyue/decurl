# API Overview

The application-facing Decurl API uses `@decurl/react-router` as the main entry.

Besides the default entry, `@decurl/react-router` exposes `configured`, `codec`, `decode`, `routeSpec`, and `pagination` sub-entries through conditional exports.

## Entries

| Package | Entry | Purpose |
| --- | --- | --- |
| `@decurl/react-router` | `@decurl/react-router` | Ready-to-use React Router hooks |
| `@decurl/react-router` | `@decurl/react-router/configured` | Create store-bound hooks and Provider |
| `@decurl/react-router` | `@decurl/react-router/codec` | FieldCodec, Search Fields, URLSearchParams codec |
| `@decurl/react-router` | `@decurl/react-router/decode` | Decode primitives and composition helpers |
| `@decurl/react-router` | `@decurl/react-router/routeSpec` | Define path and search contracts, then generate type-safe hrefs |
| `@decurl/react-router` | `@decurl/react-router/pagination` | Page, pageSize, and pagination behavior |
