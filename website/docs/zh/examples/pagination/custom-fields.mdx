# Custom Fields

这个示例向 `useSearchPagination(fields)` 传入自定义 Fields，使用 `p`、`size` 作为 URL key，并把 page size 限制为 `6`、`12`、`24`。

自定义 Fields 场景应让 codec 和选择器消费同一份本地 `pageSizeOptions`。`useSearchPagination.pageSizeOptions` 只属于默认分页模型，不应在这里使用。
