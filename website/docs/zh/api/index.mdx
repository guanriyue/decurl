# API 概览

Decurl 面向应用开发者的 API 以 `@decurl/react-router` 为主入口。

除了默认入口，`@decurl/react-router` 还通过 conditional exports 暴露 `configured`、`codec`、`decode` 和 `pagination` 子入口。

## 入口

| 包 | 入口 | 用途 |
| --- | --- | --- |
| `@decurl/react-router` | `@decurl/react-router` | 开箱即用的 React Router hooks |
| `@decurl/react-router` | `@decurl/react-router/configured` | 创建绑定 store 的 hooks 与 Provider |
| `@decurl/react-router` | `@decurl/react-router/codec` | FieldCodec、Search Fields、URLSearchParams codec |
| `@decurl/react-router` | `@decurl/react-router/decode` | decode primitive 与组合工具 |
| `@decurl/react-router` | `@decurl/react-router/pagination` | page、pageSize 与分页联动行为 |

1. 先读 [快速开始](/zh/guide/start/getting-started)。
